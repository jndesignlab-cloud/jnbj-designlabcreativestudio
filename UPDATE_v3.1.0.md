# v3.1.0 — Interactive Game Profile

## Homepage changes
- Replaced the conventional portfolio pitch with a game-inspired player profile.
- Added a character card, level badge, status HUD, and availability bar.
- Added four clickable profile statistics with interactive modal details.
- Added a clickable skill tree with five creative capabilities.
- Shortened the homepage wording and reduced the conventional sales-pitch tone.
- Preserved the selected work, current collaborations, field notes, testimonials, and contact sections.

## Interaction
- Stat and skill cards are keyboard accessible.
- Modal closes through the close button, backdrop, or Escape key.
- Focus returns to the selected stat card after closing.
- Mobile layouts use a single-column profile and stat interface.

## System compatibility
- Project loading, project preview modal, inquiry system, tracking, email system, CV, services, and Apps Script integration remain unchanged.
- No Code.gs update or Apps Script redeployment is required.
