Recommended project image format

Best preview size:

Use 16:9 images such as 1920×1080, 1600×900, or 1280×720.

Image hosting:

You can use GitHub images, Google Drive direct image links, or any public image URL.

Project card display:

The homepage shows up to 6 projects marked as Featured. The full projects page shows all published projects.

Gallery images:

Paste one image URL per line. Recommended size is 1920×1080 or 1600×900.

Skills / Services Used:

Separate items using hyphens, commas, or new lines. Example: Social Media Management - Graphic Design - Campaign Posters.

Case study fields:

Use Problem / Goal, What I Did, and Result / Outcome to make each project page feel more professional and recruiter-friendly.

Featured projects:

Only the first 6 projects marked Featured will appear on the homepage. A project can belong to multiple filter categories on the full projects page. Example: DesignLab and Social Media.
