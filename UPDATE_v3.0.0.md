# DesignLab Portfolio v3.0.0

## Portfolio Art Direction Cleanup

This is an experimental visual redesign built on top of the complete and working v2.10.0 system.

### Main changes

- Rebuilt the homepage around a more personal, editorial introduction.
- Made Jann Nathaniel Jaravata and the relationship to DesignLab Creative Studio immediately clear.
- Reduced rounded cards, pill styling, glow effects, gradients, and visible system controls.
- Removed the public theme selector and visitor counter from the interface.
- Standardized the public portfolio in a restrained DesignLab Blue visual identity.
- Reworked selected projects into an asymmetrical editorial grid.
- Replaced generic positioning copy with specific descriptions of actual work and workflow.
- Simplified the homepage structure to Work, About, Ongoing Work, Studio Notes, Feedback, and Contact.
- Preserved the complete inquiry, tracking, CV, Services, Projects, Insights, and admin systems.
- Added keyboard access to homepage project previews.
- Fixed the duplicate Back button in the final inquiry-form step.

## Apps Script

No Apps Script update is required. The included `Code.gs` is unchanged from the working v2.10.0 system.

## Recommended review

Treat this as an experimental v3 direction. Compare it with the v2.10.0 stable backup before replacing the public website.
